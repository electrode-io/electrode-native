## `ern platform use <version>`
#### Description
* Switch to (or activate) a specified platform version  
* Run the the `ern platform install` command if the version is not already installed

#### Syntax
`ern platform use <version>`

#### Remarks
* If the target platform version is not installed locally, the `ern platform use <version>` command runs the [ern platform install] command which installs the versionâ€”before switching to the specified version.
